Vimeo Video Viewer

Documentation for the Vimeo Video Viewer project.

Introduction

This project allows you to embed and interact with Vimeo videos using a custom video viewer.

Getting Started

To use the Vimeo Video Viewer, follow these steps:

  1. Clone the repository from GitHub.
  2. Navigate to the project directory.
  3. Open src/index.html in your browser to view the video player.

Usage

To embed a Vimeo video, modify the iframe code in src/index.html with your desired video URL.

Customization

You can customize the styles by editing the src/css/styles.css file.
